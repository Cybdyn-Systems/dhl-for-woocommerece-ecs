## üìñ Description

The plugin is compatible with the following DHL service offerings depending on your origin country:
- **DHL eCommerce Asia:** AU, CN, HK, IN, VN, TH, SG, MY

## ‚ú® Features

- **Fast and Easy Label Creation** - Generate shipping labels for your national and international orders with ease.
- **Automatic Tracking Codes** - Automatically receive a tracking code for each label created.
- **Additional Delivery Services** - Access additional services such as shipment insurance.
- **Return Parcel Handling** - Print return labels with a return address, making it easy for customers to return.
  shipments

## üöÄ Installation & Configuration

1. Download the plugin zip file from the GitHub repository.
2. In WordPress Admin, navigate to **Plugins ‚Üí Add New** and click **Upload Plugin**.
3. Upload the zip file directly and click **Install Now**.
4. Activate the plugin through the **Plugins** screen.
5. Navigate to **WooCommerce ‚Üí Settings ‚Üí Shipping** to configure DHL settings.
